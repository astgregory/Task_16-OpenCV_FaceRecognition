# Эксперимент по размытию лица с распознаванием элементов

Проект демонстрирует обработку изображений: распознавание лица и глаз на фото, размытие лица (кроме глаз) и наложение виртуальных очков.

## Функциональность

Программа выполняет следующие шаги:
1. Загружает изображение актёра/актрисы.
2. Распознаёт лицо с помощью каскада Хаара и выделяет его овалом.
3. Распознаёт глаза внутри области лица и выделяет их (в расширенной версии — накладывает солнцезащитные очки).
4. Размывает всё лицо кроме глаз с помощью функции `blur_face`.
5. Визуализирует исходное и обработанное изображение.


## Ключевые компоненты

* **Распознавание лиц и глаз:** каскады Хаара (`haarcascade_frontalface_default.xml`, `haarcascade_eye.xml`).
* **Выделение лица:** овал, нарисованный через `cv2.ellipse()`.
* **Размытие:** функция `blur_face()` на основе `cv2.GaussianBlur()`.
* **Наложение очков:** масштабирование и позиционирование PNG‑изображения с прозрачностью.
* **Визуализация:** `matplotlib.pyplot` для отображения результатов.

## Используемые библиотеки

* `cv2` (OpenCV) — обработка изображений, распознавание, рисование.
* `gdown` — загрузка изображений и ресурсов по URL.
* `numpy` — работа с массивами пикселей.
* `matplotlib` — визуализация результатов.

## Функция размытия

```python
def blur_face(img):
    (h, w) = img.shape[:2]
    dW = int(w / 3.0)
    dH = int(h / 3.0)
    if dW % 2 == 0:
        dW -= 1
    if dH % 2 == 0:
        dH -= 1
    return cv2.GaussianBlur(img, (dW, dH), 0)
